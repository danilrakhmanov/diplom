# SportStore IS

Дипломный проект на тему: **«Разработка информационной системы магазина спортивных товаров»**.

Система помогает вести каталог спортивных товаров, учет клиентов, оформление продаж, контроль складских остатков и базовую аналитику.

## Возможности

- просмотр сводных показателей магазина;
- каталог товаров с поиском и фильтрацией по категориям;
- добавление новых товаров и новых категорий;
- клиентская база с бонусными баллами;
- оформление заказа с автоматическим уменьшением остатков;
- история продаж;
- список товаров с низким остатком на складе.

## Запуск

Требуется Python 3.10 или новее. Внешние библиотеки не нужны.

```powershell
python app.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:8000
```

При первом запуске создается файл базы данных `sportstore.db` и заполняется демонстрационными данными.

## Сущности базы данных

- `categories` — категории спортивных товаров;
- `products` — товары, цены, бренды и складские остатки;
- `customers` — покупатели и бонусные баллы;
- `orders` — заказы покупателей;
- `order_items` — состав заказа.
